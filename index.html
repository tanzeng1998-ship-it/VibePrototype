🤖 Agent

智能体（Agent）

具备自主规划、推理与执行能力的 AI 助手。能够理解复杂目标、拆解任务步骤、调用工具并完成端到端的工作流程。

🧩 Plugin

插件（Plugin）

Agent 能力的模块化扩展。每个插件封装特定功能或集成第三方服务，Agent 可按需调用。

⚡ Skill

技能（Skill）

领域知识、标准化工作流和专用工具集的合集。每个 Skill 聚焦单一场景，如代码开发、内容创作、数据分析等。

🔗 MCP

模型上下文协议

连接 AI 与外部工具、数据源的标准通信协议。定义 AI 调用工具、获取上下文、管理资源的统一规范。

📋 Rules

规则（Rules）

用户为 AI 行为设定的约束与指导框架。通过预定义的工作边界、偏好设置和质量标准，确保 AI 输出符合预期。

⌨️ Command

指令（Command）

用户与 AI 交互的触发机制。通过自然语言或预设指令格式，即时唤醒特定功能、切换工作模式或调用工具链。
